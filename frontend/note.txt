Cách chạy (local, trong PyCharm)

Frontend (React + Vite)

1. Cài Node (nên Node 18+).

2. Vào frontend/, cài dependencies và chạy dev:

cd frontend
npm install
npm run dev


Vite sẽ serve tại http://localhost:5173 (hoặc port khác hiển thị lên console).

Trong PyCharm:

PyCharm Pro hỗ trợ Run Config cho npm; nếu bạn dùng PyCharm Community, mở terminal tích hợp và chạy npm run dev. Browser sẽ mở cổng Vite; UI sẽ fetch http://127.0.0.1:5002/api/....
